🔺 Projet Orgonites Énergétiques

Un univers entre énergie, géométrie sacrée et design hacker 🧿⚡

🔧 Matériel nécessaire

  • 🧪 Résine époxy ou polyester (transparente de préférence)
  • 🧲 Fil de cuivre (0.5 à 1 mm), éventuellement dénudé
  • 🔮 Pierres semi-précieuses : quartz, améthyste, citrine, tourmaline noire
  • 🌈 Poudres métalliques (aluminium, laiton, fer)
  • 🎨 Pigments naturels ou mica pour la coloration
  • 📐 Moules en silicone : pyramide, cône, disque, pendentif
  • 🧤 Gants + masque + support de travail ventilé
💡 Astuce : Tu peux recycler des copeaux de métaux (perçage) ou du cuivre de câbles usagés.

💎 Composition énergétique

  • 🌟 Résine époxy transparente (base vibratoire)
  • 🧲 Fil de cuivre (conducteur et générateur de spirales)
  • 🔮 Cristaux : Quartz, Améthyste, Citrine, Tourmaline noire
  • 🌈 Pigments naturels, poudres métalliques (effets de couches)

🔄 Formes de fil de cuivre recommandées

✅ Spirale Fibonacci : Pour amplifier l'énergie selon les lois naturelles.

✅ Tresse hélicoïdale (double spirale) : Pour équilibrer les polarités Yin/Yang.

🧘 Couleurs & Chakras

Chaque couche peut être associée à un chakra :

  • 🔴 Rouge (racine) : grenat, jaspe
  • 🟠 Orange (sacré) : cornaline
  • 🟡 Jaune (plexus) : citrine
  • 💚 Vert / 💗 Rose (cœur) : aventurine, quartz rose
  • 🔵 Bleu (gorge) : lapis lazuli
  • 🟣 Violet (3e œil, couronne) : améthyste, quartz clair

⚙️ Structure d'une pyramide

Une idée de configuration :

  1. Base : Métaux broyés + Tourmaline noire
  2. Spirale de cuivre centrale
  3. Quartz au centre, entouré de pierres secondaires
  4. Couche de résine pigmentée chakra
  5. Pointe : Cristal maître

function draw() {
  translate(width/2, height/2);
  beginShape();
  for (let a = 0; a < TWO_PI * 5; a += 0.1) {
    let r = 5 * a;
    let x = r * cos(a);
    let y = r * sin(a);
    vertex(x, y);
  }
  endShape();
  noLoop();
}
